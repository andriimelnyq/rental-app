# RENTAL APPLICATION

[DEMO](https://rental-app-9efce.web.app/)

### Used technologies
- React.
- TypeScript.
- SCSS.
- HTML.
- Redux (Redux-Toolkit).
- Material UI.
- Leaflet.
- Firebase.

### Instructions
For run it on your machine: just `fork` this repo. Then `clone` it to your computer and run `npm install` (npm i). After use `npm run dev`.
